Speaker

Cassie Schmitz / Co-Founder, PrintWhatYouLike.com

Topic

Rails 3

After the long wait, Rails 3 Beta has finally been released! With the merge with Merb, Rails has undergone significant changes. In this talk, Cassie will go over what's new in Rails 3 including:

  • Controller Abstraction
  • Routing with ActionDispatch
  • Model Abstraction
  • Arel Integration
  • Revamped ActionMailer
