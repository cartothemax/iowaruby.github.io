Topic

JRuby

JRuby is a pure-Java implementation of Ruby. It is Ruby for the JVM.

Besides being a rock-solid implementation of Ruby (some say JRuby is currently the best Ruby available [1]), JRuby also provides integration between Ruby and the JVM.

We'll discuss and see demos of the following

  • Running Java code from JRuby
  • Running Ruby code from Java (embedding JRuby)
  • JRuby Compiler
  • Deploying Ruby web applications as WAR files
  • Testing Java code using Ruby test frameworks
  • JRuby / Ant integration

[1] http://www.infoq.com/interviews/Yehuda-Katz-on-Rails-3.x
