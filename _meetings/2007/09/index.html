September 2007

Thursday, September 20, 2007 - Organizational Meeting

6:00 - 7:30 PM

Please join us for our first meeting. We're just getting started, so please let us know how we can best serve the needs and interests of the Iowa Ruby community.

Speaker

David W. Body / Big Creek Software, LLC

Topics

  • Organizational Meeting
  • Getting Started with Ruby

Agenda

  • Welcome
  • Introductions
  • Getting Started with Ruby

We'll cover how to install Ruby and how to get started using it.

We'll introduce several common Ruby tools including irb, ri, and rdoc.

We'll talk about anything else of interest to the group, so please join us!
