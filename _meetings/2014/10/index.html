October 2014

October 21, 2014

6:00 - 7:30 PM

Topics & Speakers

This month we'll have two shorter topics (one for Ruby newbies and one slightly more advanced) that are somewhat related and should help us focus on thinking functionally.

Enumerable Module

Dan Buettner / Merchants Bonding Company

First, Dan Buettner will introduce the Enumerable module which includes handy methods like each, map, reduce, count, min, max, any?, all?, and so on. We'll see how easy it is to leverage Enumerable to unlock the power of Ruby.

Transducers

David W. Body / Big Creek Software, LLC

Second, David W. Body will cover Transducers, which are "composable algorithmic transformations." This is a relatively new programming construct that is being introduced in the forthcoming Clojure 1.7 but has already been ported to several other languages including Ruby. We'll try to understand what problem transducers are meant to solve and discuss whether we think they are useful in Ruby.
