April 2014

April 15, 2014

6:00 - 7:30 PM

Topic

Round-table Discussion: Ruby Debugging

Inspired by a recent episode of the Ruby Rogues podcast, we'll have a an open round-table discussion of Ruby debugging and debugging software in general. This will be primarily a group discussion, so come prepared with any debugging questions or techniques you'd like to share.

Topics to be discussed may include some or all of the following:

  • Reading error messages and stack traces
  • Using puts or logging statements
  • Experimentation / hypothesis testing
  • Unit tests & integration tests as debugging aids
  • Using source code revision history
  • Effective use of search engines
  • Ruby debuggers
  • What to do when a bug appears in production but not development or test environments

There will be plenty of time for questions and discussion.

Please join us no matter what your level of Ruby expertise might be!
