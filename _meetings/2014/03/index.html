March 18, 2014

6:00 - 7:30 PM

Topic

Back to Basics - Ruby Language

This month we'll go back to basics and focus on the Ruby language.

We'll cover several Ruby language features with code examples of each. Specific language features will be chosen so everyone from complete beginners to experienced Rubyists will have a chance to learn something.

Topics to be discussed may include some or all of the following:

  • Ruby equality: ==, eql?, equal?, ===
  • The Symbol.to_proc trick (&:my_method)
  • Keyword arguments (Ruby 2)
  • Lazy enumerators (Ruby 2)
  • Infinity in Ruby

There will be plenty of time for questions and discussion.

Please join us no matter what your level of Ruby expertise might be!
