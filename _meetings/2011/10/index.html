Speaker

Scott Robinson / Merchants Bonding Company

Topic

Using the state_machine gem

A finite-state machine or "state machine" is a common behavioral model used in computer programming. In a nutshell, a state machine can be in one of a finite number of states. An event or condition can then trigger a transition to another (or the same) state, and various actions can occur during a transition. (See Finite-state machine for more.)

Scott will discuss the state_machine Ruby gem, including

  • Why you might want to consider using the state_machine gem
  • Summary of functions/options provided by the gem
  • How we're using the state_machine gem at Merchants Bonding
