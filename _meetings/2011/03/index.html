March 2011

March 15, 2011

6:00 - 7:30 PM

Topic

Working Effectively with Third-Party Web Services

Speaker

David W. Body / Big Creek Software, LLC

David has been developing a Rails application whose core functionality involves interacting with a third-party commercial ASP.NET XML web service. The main interaction involves posting an XML request to the web service, and then asynchronously receiving XML results which may come hours or days later. The web service is production-only, involves sensitive data, and there is no separate test environment.

David will demo the application and then we'll dive into the source code.

Topics will include

  • Development, testing, and deployment strategies
  • Simulating a web service with Sinatra
  • Dealing with XML documents whose elements may not all be known in advance

This Rails application is under active development, and David would welcome suggestions for changes and improvements.

There will be plenty of time for discussion and questions.
