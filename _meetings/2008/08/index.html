6:00 - 7:30 PM

Speaker

Frank Luke / Big Creek Software, LLC

Topic

NetBeans

InfoWorld just published a review of nine Rails IDEs and editors, and rated NetBeans 6.1 the highest of the bunch.Â  NetBeans, which is free and runs on Windows, Linux, Mac OS X, and Solaris, even beat TextMate, the editor of choice of the Rails core team.

Frank will demonstrate the Ruby and Rails features of the latest version of NetBeans and will describe his experience using NetBeans as a Ruby IDE on a daily basis for several weeks.

NetBeans Ruby features include:

  • Ruby project management
  • Ruby source code editor with code completion
  • Ruby refactoring support
  • Support for Test::Unit, AutoTest, and RSpec
  • Support for running Rails generators and Rake tasks within the IDE
  • Ruby and Rails Debugger
