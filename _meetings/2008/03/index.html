March 2008

Thursday, March 27, 2008*

6:00 - 7:30 PM

Speaker

Luke Amdor / Iowa Student Loan

Topic

Alternative Ruby (JRuby / Rubinius / IronRuby)

*Note:  This is a special meeting date.  We will return to our normal third Thursday meeting date in April.

Lately there has been a plethora of news surrounding alternative Ruby interpreters: Sun hires the main JRuby developers, Oracle begins deploying JRuby on Rails applications, Engine Yard hires Rubinius developers to work full time on Rubinius, and Microsoft is currently developing it's own Ruby interpreter IronRuby to run on its CLR.

In this presentation, we'll talk about the motivation behind these interpreters, where they're headed, and what impact this is having on the Ruby community.

As far as practical examples go, we'll be showing an emphasis on JRuby and JRuby on Rails.  We'll show how to distribute Ruby scripts with only Java being installed on the platform, how to deploy an existing Ruby on Rails application into Sun's Glassfish, a JavaEE application server, how to create a quick and dirty Swing app using Ruby.

Topics:

  • Motivation / Extending Ruby's Reach
  • Sucking the information out of Matz's brain / Specs
  • Rubinius
  • Eating your own dog food / Ruby in Ruby
  • JRuby / Introducing Ruby to the enterprise
  • Deploying RoR applications in a business environment
  • IronRuby
