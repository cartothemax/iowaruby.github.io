Speaker

David W. Body / Big Creek Software, LLC

Topic

Capistrano

Capistrano, a tool for automating tasks on remote servers, has become the de facto standard for deploying Rails applications.  But it can be used for many other things, including deploying applications written in other languages and frameworks, as well as general system administration.  David's presentation will cover the following topics:

  • Getting started with Capistrano
  • Capistrano's requirements and assumptions
  • Writing Capistrano recipes
  • Deploying Rails applications with Capistrano
  • Rolling back changes on remote servers
  • Using Capistrano for system administration and other tasks
