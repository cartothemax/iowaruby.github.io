September 2008

Thursday, September 11, 2008

6:00 - 7:30 PM

Speaker

Cassie Schmitz / Iowa Interactive

Topic

RSpec

RSpec is a behavior-driven development framework for Ruby. Cassie will demonstrate the features of RSpec and describe her experience with using RSpec in her latest Ruby on Rails project.

Topics that will be covered:

  • Behavior-Driven Development vs. Test-Driven Development
  • Expressing the expected behavior of your code
  • Spec::Rails
  • Story Runner
  • Mock objects and method stubs
  • Generating specifications
  • RCov Coverage tool integration
