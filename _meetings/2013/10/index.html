October 2013

October 15, 2013

6:00 - 7:30 PM

Speaker

Luke Amdor / Banno

Topic

Writing Internal CLI Tools using Ruby and Thor

At Banno, we have built up an internal tool called banno-utility-belt or bub for short. This tool allows all developers and ops to have a singular go-to tool for many of our commonly done tasks. Using Thor and Ruby has allowed us to have a extendable and self-documenting command line interface.

In this presentation, you'll learn how to leverage Ruby and Thor to codify processes and practices for the betterment of your internal ecosystem.
