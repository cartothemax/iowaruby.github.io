Speaker

Topic

Chef 101

Chef is a configuration management platform written in Ruby. With Chef you can build up a server from a base image to its desired production ready state. Chef allows you to express the state of your infrastructure as code, which can be reviewed, tested, and version controlled. This allows for changes in said infrastructure to be applied with much lower risk. The nature of Chef allows engineers to bring up parts or the whole infrastructure with tools like Vagrant. Unit testing with chefspec and integration testing with test-kitchen are new developments in the Chef Community.

This talk to will cover the following tools:

  • Chef
  • Knife
  • Berkshelf
  • Vagrant
  • Chefspec
  • Test-Kitchen

Nic Grayson is Technical Operations Engineer at Banno.
