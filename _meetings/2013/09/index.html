September 2013

September 17, 2013

6:00 - 7:30 PM

Topic

Getting Started with Ruby

This month we're going back to basics with an introductory level discussion on getting started with Ruby. We'll discuss the proper way to set up an environment for Ruby programming, including

  • Installing Ruby (including ways not to do it)
  • Installing and managing Ruby gems
  • Using tools like Bundler and Rake
  • Other elements of a good Ruby development environment

We'll assume the typical attendee is a programmer with experience in at least one other language but little to no experience with Ruby. Nevertheless, everyone from complete beginners through experienced Rubyists is invited to attend and might learn something.
