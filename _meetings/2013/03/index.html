Topic

RubyMotion

RubyMotion is a set of tools that allows you to create native iOS applications for iPhone or iPad using the Ruby language.

A few major points from the RubyMotion website:

  • Command-line interface for creating and maintaining apps.
  • Build apps using Rake.
  • Based on MacRuby codebase, developed over 5 years at Apple.
  • Produces compiled, native iOS apps.
  • Full access to all Objective-C APIs.
  • 100% compliant with Apple's App Store guidelines.

Darrin is a Des Moines area software developer currently working for Lean TECHniques and a frequent speaker at the Iowa Ruby Brigade.
