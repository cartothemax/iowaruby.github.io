Speaker

Brandon Ruschill / Goodsmiths

Topic

Writing Custom Rack Middleware

Brandon Ruschill will give us an introduction to writing custom Rack middleware.

Rack provides a minimal interface between webservers supporting Ruby and Ruby frameworks. It sits between a webserver such as Apache or nginx (via Phusion Passenger) and a Ruby web framework such as Sinatra or Ruby on Rails. Between the server and the framework, Rack can be customized using middleware.

Brandon is a software developer at Goodsmiths in West Des Moines.
