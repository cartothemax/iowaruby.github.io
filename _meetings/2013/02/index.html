February 2013

February 19, 2013

6:00 - 7:30 PM

Speaker

Toran Billups / The IMT Group

Topic

Ember.js

With all the hype around javascript these days it's hard to know what library or framework will help you deliver more features with less code.

If you are interested in learning more about a full featured framework that provides an application programming model similar to the server side MVC frameworks you already know and love, maybe ember.js is for you.

Join me for a discussion about why I decided it was time to use a real framework on the client side and how it changed the way I write javascript today.

Some knowledge of javascript is recommended but not required.

Toran Billups is a Python and JavaScript enthusiast with hopes of knowing enough about ember.js and django to build a modern web app.
