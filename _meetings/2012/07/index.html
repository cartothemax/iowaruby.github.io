Speaker

Dan Buettner / Merchants Bonding Company

Topic

JavaScript Testing with Jasmine

If you're like me, you've struggled with adding custom JavaScript functionality to your application because there wasn't a good way to test your code besides constantly reloading your browser page (and emptying your cache!).

Enter Jasmine, an RSpec-like tool for automated testing of your JavaScript code.

We'll look at the basics of setting up Jasmine in a Rails project, its RSpec-like syntax, and talk about a couple of gotchas that are easy to work around ... provided that you know about them.

This talk will be great for JavaScript beginners, as that's really what I am - but the installation and setup of Jasmine should be useful for seasoned pros as well.

About Dan:

Dan works at Merchants Bonding Company full-time as a web application developer; in his copious spare time, he is president of Port 80 Productions, a small (very small) web development shop.

In a past life, he worked as a software developer and IT manager at The Des Moines Register, Outsell Inc., and Gannett Co.

He enjoys running, bicycling, and talking about himself in the third person.
