6:00 - 7:30 PM

Speaker

Dan Buettner / Merchants Bonding Company

Topic

Full-Text Search Using Solr

Search functions on the web are ubiquitous, but not all of them work well enough to actually be useful. If you've wondered how to improve your own app's capabilities, come learn about Solr - a powerful, easy to use, and fast full-text search solution that easily plugs into a Rails application.

We'll talk about how to make your models searchable, boosting results based on where search terms appear, Solr configuration for synonyms, and constructing "advanced search" combinations of parameters.

If you're cobbling together your own full-text-like search routines against SQL databases in Rails by splitting search strings into tokens and then concatenating SQL snippets, you're working too hard. (Go ahead, ask me how I know).

Dan Buettner is a web developer at Merchants Bonding Company in Des Moines and president of Port 80 Productions. He excels at merge commits and writing SQL by hand.
