November 2012

November 20, 2012

6:00 - 7:30 PM

Speaker

Collin Miller / @collintmiller

Topic

Concurrency for Humans

The erstwhile Google Wave promised multi-user editing of documents for all. Wouldn't it be nice if it was a Ruby library and worked for arbitrary JSON documents? Tonight we'll explore a port of the excellent ShareJS library for Ruby.

Collin Miller is both a native Iowan and a native Rubyist. While he isn't programming Collin enjoys performing improvised comedy on stage with his troupe, "The Beards".
