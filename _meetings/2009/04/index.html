April 2009

April 9, 2009

6:00 - 7:30 PM

Speaker

Cassie Schmitz / PrintWhatYouLike.com

Topic

PrintWhatYouLike.com

In this presentation, Cassie will talk about building her new startup, PrintWhatYouLike.com, an online editor that lets you reformat any web page for printing. Try it out at: www.printwhatyoulike.com Cassie will show how cool Ruby technologies have been used to power PrintWhatYouLike.

Topics that will be covered:

  • Reasons for moving from Google App Engine to Ruby on Rails
  • Creating a proxy server
  • Using Hpricot to parse even horribly invalid HTML
  • Supporting web pages from around the world
  • Incorporation of Twitter on homepage
  • and more...
