Speaker

Luke Amdor / Big Creek Software, LLC

Topic

Creating Mac OS X Applications with MacRuby

MacRuby is a full 1.9 compatible version of Ruby that runs on the ObjectiveC runtime. But that's not all. MacRuby brings the elegance and flexibility of Ruby to create full fledged Mac OS X desktop applications. MacRuby doesn't sacrifice speed to do this either.

In this presentation, we'll discuss:

  • how MacRuby fits in the Mac OS X and Ruby ecosystem.
  • Ruby, Objective-C, and CoreFoundation fundamentals.
  • how MacRuby works its magic
  • how to set up MacRuby projects
  • integration with Interface Builder
  • future of MacRuby

We'll also put together some Mac OS X applications:

  • the quintessential Twitter client
  • a Flickr photo search using HotCocoa
  • using Test Driven Development to create Cocoa applications

We'll also explore the possibility of iPhone applications using MacRuby.
