Speaker

Darrin Holst / GeoLearning

Topic

Shoes

Shoes is a lightweight Ruby GUI toolkit for beginners developed by Why the Lucky Stiff. Here's an example of a simple Shoes application:

Shoes.app {
 @push = button "Push me"
 @note = para "Nothing pushed so far"

 @push.click {
   @note.replace "Aha! Click!"
 }
}

Darrin will present an introduction and overview of Shoes, including the recently released "Raisins" version.

For more information on Shoes, see

http://shoooes.net/about/

http://github.com/why/shoes/tree/master
